AI 튜터

AI 튜터는 24/7 이용 가능한 맞춤 가상의 튜터가 학습 질문에 즉시 응답하고, 개인 맞춤형 대화 학습을 제공합니다. 학생이 자신의 페이스에 맞게 학습할 수 있도록 도와주고, 각 학생의 강점과 약점을 파악하여 최적의 학습 경험을 제공합니다.

주요 기능

  • 개인화된 학습 경로 제공 - 학생의 수준과 목표에 맞는 맞춤형 학습 경로 설계
  • 실시간 질의응답 서비스 - 언제 어디서나 학습 관련 질문에 즉시 답변
  • 맞춤형 학습 자료 추천 - 학생의 관심사와 학습 스타일에 맞는 자료 제공
  • 학습 진도에 맞는 피드백 - 각 학습 단계별 세밀한 피드백과 교정 제공
  • 다양한 학습 스타일 지원 - 시각적, 청각적, 읽기/쓰기, 운동감각적 학습 방식 모두 지원
  • 주기적인 복습 알림 - 망각 곡선을 고려한 최적의 복습 시점 알림
  • 학습 동기 부여 - 개인화된 목표 설정과 성취 축하 메시지 제공
